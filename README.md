# Python_Game

Python Adventure is a small, browser-based game that teaches Python concepts through quests, puzzles, and a turn-based “code-as-spells” system.

What you need to install

Node.js LTS (>= 18) — https://nodejs.org

Git — https://git-scm.com

VS Code - https://code.visualstudio.com

ESLint (dbaeumer.vscode-eslint)

Prettier – Code formatter (esbenp.prettier-vscode)

EditorConfig for VS Code (EditorConfig.EditorConfig)

GitLens (eamodio.gitlens)

Error Lens (usernamehw.errorlens)

Path Intellisense (christian-kohler.path-intellisense)

vscode-icons (vscode-icons-team.vscode-icons) or Material Icon Theme
